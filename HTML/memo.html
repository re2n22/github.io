1. HTML 기본 태그로 글 작성하기

html은 태그로 작성함
태그들을 열고 닫은 후 내부에 글, 그림등을 넣을 수 있음.
태그들은 종류와 용도가 매우 다양하며 매우 많이 존재.
기본적인 태그들을 알아보자

----------------------------

<p>글 본문</p>
<h1>글 제목</h1>
<img src="이미지 경로">
<a href="링크주소">하이퍼링크</a>
<button>버튼</button>
<ul>리스트<li> </li>리스트</ul>
<ol>리스트<li> </li>리스트</ol>

 이처럼 용도에 맞게 태그를 사용하면 된다.

2. 기본적인 웹 페이지 스타일링

간단히 스타일링을 넣는법

<p style="font-size: 10px; color: red"> 글자 </p>

 거의 모든 태그는 style=""이라는 속성을 열 수 있음.
거기 안에 스타일이름:스타일 값; 형식으로 스타일 적용가능
여러개를 넣을 때는 쭉 스타일을 적되 세미콜론을 뒤에 써야함.

자주 사용하는 글자 스타일들

font-size: 20px;
font-family: 'gulin';
color: black;
letter-spacing: -1px;
text-align: center;
font-weight: 500;

 이미지 정렬과 폭 조정

display: black;
margin-left: auto;
margin-right: auto;
width: 150px;

 이미지는 width를 많이 씀. px또는 %로 폭 조정 가능.
이미지를 가운데 정렬하고 싶으면
display: block; margin-left: auto; margin-right: auto를 기입할 것.

텍스트의 일부만 스타일을 변경하고 싶을 때는 <span>태그로 감싼 뒤 스타일을 따로 주면 된다.
<strong>태그는 글자굵기를 굵게 표현가능

(참고)span태그는 display:inline이라는 스타일속성을 내포하고 있으며
display: inline을 가지고 있는 요소는 폭, 높이등을 설정할 수 없음.
폭이나 높이를 주고싶다면 span을 감싸고있는 p에 줄 것.

3. CSS파일 만들고 첨부하는 법

style 속성 안에 들어갈 코드가 길어지면 html이 더러워지기 때문에 이걸 따로 작성 할 수 있음. CSS라는 파일로 빼면 됨.

CSS파일을 사용하려면 CSS파일을 만든 후에 html에 첨부해야함.
먼저 작업폴더에 .css로 끝나는 파일을 만든 후에 html파일로 돌아와서

<link href="~~css파일경로" rel="stylesheet">

 head태그 안에 link태그를 넣고 href라는 속성에 css파일 경로를 지정해놓으면 css파일 이용가능.
이제 css파일에서 스타일작성이 가능. 정확한 방법은

1. css파일에 스타일을 집어넣고 .어쩌고 작성

.profile{
width: 200px;
height: 200px;
display: block;
margin: auto;
}

 스타일을 적고 .어쩌고 대충 작명하고{}중괄호로 묶기.
위와 같은 형태를 클래스라고 부름.

2. html에 클래스명을 첨부하여 스타일을 넣을 수 있음.

<img src="kyaru.jpg" class="profile">

 class속성을 열고 클래스명을 집어넣으면 css파일에서 묶어놨던 스타일들이 전부 적용됨.

3. brackets에디터에선 클래스작성과 편집이 편함.
class대충 작성하고 command+e하면 바로 열림.

class말고 다른 식으로도 스타일을 묶을 수 있는데

.class {width: 200px} // 클래스
 #special {width: 200px} // 아이디
 div {width: 200px} // 태그
CSS selector라고 칭하는것들.
클래스 selector는 클래스명{}이렇게 적고 class="클래스명"을 가진 요소에 스타일 적용가능
아이디 selector는 #아이디{}이렇게 적고 id="아이디명"속성을 가진 요소에 스타일을 적용가능.
태그 selector는 div{스타일~~}이렇게 적고 모든div태그에 스타일을 적용가능.
.은 클래스, #은 아이디

셀렉터의 우선순위도 따로 있음.

style""(1000++)
#id(100++)
.class(10++)
tag(1)
숫자가 높을수록 우선 적용됨.

4. div를 이용한 네모박스 디자인

네모박스 디자인에 많이 사용하는 css속성들

.box{
margin: 20px;
padding: 30px;
border: 1px solid black;
border-radius: 5px;
}

 margin은 바깥 여백,
padding은 안쪽 여백,
border는 테두리(순서대로 두께, 선의 종류, 색상)
border-radius는 테두리 둥글게 처리.
가운데 정렬도 많이 하지만 저번이미지처럼 마진오토주면 됨.

마진과 패딩은 원하는 방향에만 줄 수 있음.

.box{
margin-top: 20px;
padding-top: 20px;
}

 top, left, bottom, right로 원하는 방향에 원하는 만큼 여백을 줄 수 있음.
(참고)마진은 음수도 사용가능. -20px이런식으로.
(참고)margin:1px 2px 3px 4px; 이렇게 띄어쓰기를 이용해 작성하면 차례대로 상 우 하 좌 마진을 한번에 줄 수 있음.

display: block이 내장되어있는 div박스

.box{
display: block
}

 모든 div, p, h1, li등은 display: block속성을 기본으로 가지고 있음.
그래서 p, div태그를 그냥 사용하면 한 행을 전부 차지하게 됨
이게 싫다면 display속성을 다른것으로 부여할 것.
display:inline, inline-block, flex등 여러가지가 존재함

일부 스타일은 inherit됨.
font-size, color, font-family, text-align이런 속성들은 부모태그에 주면 자식태그까지 전부 상속됨.
안에 태그들이 많으면 전부 font-size를 따로 작성 할 필요가 없이 부모태그에만 주면 되서 편리함.
전부 상속되는것은 아니고 글자와 관련된 스타일들이 주로 상속됨.

5. 호환성 좋은 float

html css로 사이트 레이아웃 만들기:
만들고 싶은 레이아웃 디자인 위에 네모박스를 먼저 그리고 바깥네모부터 div써서 구현하면 됨.
어려워보이는 레이아웃도 네모박스부터 위에 그려보고 바깥네모부분부터 div로 하나하나 만들면
어떤 레이아웃이던 쉽게 만들기 가능

요소를 공중에 띄워 왼쪽/오른쪽 정렬하는 float속성

<div>
<div class="left-box"></div>
<div class="right-box"></div>
</div>

.left-box{
width: 100px;
height: 100px
float: left;
}

.right-box{
width: 100px;
height: 100px
float: left;
}

 위의 코드는 박스 두개를 만들어 각각 왼쪽으로 정렬시킴
하지만 float를 쓰면 요소를 띄우다보니 다음에 오는 html요소들이 자리를 찾지 못함.
(참고)float속성으로 가로정렬할땐 float박스들을 싸매는 하나의 큰 div박스를 만들고 폭을 지정하는것이 좋음.
그래야 모바일에서 안흘러넘침.

float를 쓰고나면 clear속성이 필요하다.

<div>
<div class="left-box"></div>
<div class="right-box"></div>
<div class="footer></div>
</div>

.footer{
clear: both
}

 clear속성을 사용하면 float다음에 오는 박스들이 제자리를 찾게 됨.
float를 썼다면 까먹지 말고 항상 넣을것.
(참고)float:none이것도 추가해주는게 나중에 버그에방차원에서 좋음.

상대적 크기단위인 %단위

.box{
width: 80%
}

 이 경우 내 부모태그의 width에 비해 80%를 차지하게 된다는 뜻.
부모태그는 나를 감싸고있는 태그를 뜻함.

6. 귀찮은 inline-block

display: inline-block 사용법

<div>
<div class="left-box"></div> <div class="right-box"></div>
</div>

.left-box{
width: 100px;
height: 100px;
display: inline-block;
}

.right-box{
width: 100px;
height: 100px;
display: inline-block;
}

 위의 코드는 박스를 만들어 왼쪽으로 정렬시키는 코드이다.
display속성만 inline-block으로 조정하면 가로로 배치가 가능하다.
inline-block은 '내 폭과 높이만큼 자리차지 하겠다'라는 뜻이다.
간편하지만 태그사이에 공백이 있다면 그대로 보여주기 때문에
가로로 정렬하려면 태그사이의 공백도 제거해주어야 한다.

공백제거 편법 1. 주석처리 기호 사용하기

<div>
<div class="left-box"></div> <!--
--><div class="right-box"></div>
</div>

주석은 실행되지 않는 코드기 때문에 주석으로 줄바꿈을 하면 된다.

공백제거 편법 2. 부모의 폰트사이즈를 0으로 만들기

<div style="font-size: 0px;">
<div class="left-box"></div>
<div class="right-box"></div>
</div>

font-size의 속성은 상속되기 때문에 안에 있는 div와 그 사이에 있는 공백도 font-size가 0p가 된다.
이러면 html은 깔끔해 질 수 있으나 css가 더러워질 수 있으니 취사선택할것
