----------------------------

display: inline-block 사용법

<div>
<div class="left-box"></div> <div class="right-box"></div>
</div>

.left-box{
width: 100px;
height: 100px;
display: inline-block;
}

.right-box{
width: 100px;
height: 100px;
display: inline-block;
}

가로로 정렬하려면 태그사이의 공백도 제거해주어야 한다.

7. 셀렉터에 대한 추가 사항

1. 강의 요약

본 강의에서는 셀렉터 사용법을 조금 더 배웁니다.

2. 강의 주요정보

셀렉터 사용법

----------------------------

특정 클래스안의 태그들 선택하기
.(임의의 클래스명) (임의의 태그명){
width:50%;
}

특정 클래스안의 직계자식만 선택하기
.(임의의 클래스명)>(임의의 태그명){
width:50%;
}

셀렉터 연달아 사용하기
.(임의의 클래스명) (임의의 태그명)>(임의의 태그명){
color: red;
}

pseudo-class에 대한 간단한 내용

<a href"#" class="link">링크</a>

.link{
text-decoration : none; 밑줄 없애기
}
.link:visited(pseudo-class){
color : black; 방문후의 링크 색
}

9. position과 좌표 레이아웃

1. 강의 요약

본 강의에서는 position에 대해 다룹니다.

2. 강의 주요정보

position 사용하기

----------------------------

.box {
position : static; /* 기준이 엄서요 */
position : relative; /* 기준이 내 원래 위치요 */
position : absolute; /* 기준이 내 부모요 */
position : fixed; /* 기준이 브라우저 창이요 (viewport) */
}
position : absolute를 가운데 정렬하기

.button {
position : absolute;
left : 0;
right : 0;
margin-left : auto;
margin-right : auto;
width : 적절히
}
